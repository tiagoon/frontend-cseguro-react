# Frontend em React

CRUD utilizando React.

#
## Packages usados
- axios
- react-router-dom

#
## Como utilizar

Para testar o projeto, execute os seguintes comandos:

Instale as dependências

```shell
npm install
```

Inicie o servidor em localhost

```shell
npm start
```

Abra a página no navegador

[http://localhost:3000](http://localhost:3000)

#
## Testes 

`npm test`

#
## Build
`npm run build`
